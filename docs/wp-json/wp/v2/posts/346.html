The following message dropped into the UK-TUG inbox this morning: a timely reminder!<\/p>\n

Dear TeX user,<\/p>\n

The upcoming EuroTeX 2009 takes place from August 31 till September 4 in Den Haag in The Netherlands. In parallel we will run the third ConTeXt conference. We already have some 25 talks and tutorials submitted but we need a few more as we run a parallel program.<\/p>\n

If you are willing to present a paper or tutorial, please let us know. The theme of the conference is \u201cEducational Uses of TeX\u201d but of course other TeX and MetaPost related topics will be covered as well. Depending on the amount of contributions we will program them in blocks that cover general topics but we will also cluster talks and turorials specific to macro packages in an additional program.<\/p>\n

You can send your proposal(s) to eurotex@ntg.nl<\/a>. Registration for the conference closes at July 1, 2009.<\/p>\n

On behalf of the organization,<\/p>\n

Taco Hoekwater & Hans Hagen<\/p><\/blockquote>\n","protected":false},"excerpt":{"rendered":"
